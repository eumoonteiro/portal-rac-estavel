# Manual de Uso - Portal Acadêmico RAC

Este documento descreve as funcionalidades e o passo a passo para utilização do portal por cada perfil de usuário: **Aluno**, **Professor** e **Coordenador**.

---

## 1. Perfil: ALUNO

### Acesso
*   **Login:** Utilize seu CPF (apenas números) e senha.
*   **Seleção de Curso:** Caso esteja matriculado em mais de um curso, selecione o desejado na tela inicial.

### Dashboard (Painel Principal)
O painel do aluno foi organizado para facilitar o acesso rápido às ferramentas mais utilizadas:

1.  **Acesso Rápido (Topo):**
    *   Botões grandes coloridos que levam a ferramentas externas (ex: Biblioteca, Drive, Sala de Aula Virtual).
    *   Basta clicar para abrir em nova aba.

2.  **Mural de Comunicados (Meio):**
    *   Visualize mensagens importantes enviadas pela coordenação.

3.  **Desempenho Acadêmico (Fim da página):**
    *   Tabela contendo suas disciplinas, notas lançadas e porcentagem de frequência.
    *   Verifique aqui sua situação (Aprovado/Reprovado).

4.  **Secretaria Virtual (Lateral Direita):**
    *   **Declaração de Matrícula:** Clique para gerar e baixar automaticamente um PDF comprovando seu vínculo.
    *   **Histórico Escolar:** Clique para baixar seu histórico de notas em PDF.
    *   **Requerimentos:**
        *   Veja o status de solicitações anteriores.
        *   Clique em **"Novo Requerimento"** para abrir um chamado para a secretaria (ex: dúvida financeira, correção de nota). Preencha Assunto e Mensagem e clique em Enviar.

## 2. Perfil: PROFESSOR

### Acesso
*   **Login:** Utilize seu CPF e senha.

### Dashboard e Ferramentas
1.  **Ferramentas de Apoio:**
    *   Links úteis configurados pela coordenação.
    *   **Botão "Minhas Turmas (Relatório)":** Clique aqui para ver seu histórico de lançamentos.
        *   Abre uma lista cronológica de todas as aulas que você lançou.
        *   Clique em **"Detalhes"** em cada linha para ver exatamente quais alunos receberam presença ou falta naquela data.

### Lançamentos (Abas Centrais)
Você verá dois cartões grandes no centro: "Lançamento de Frequência" e "Lançamento de Notas".

#### A. Como realizar a Chamada (Frequência)
1.  No cartão "Frequência", selecione a **Disciplina**.
2.  Selecione a **Data da Aula**.
3.  A lista de alunos aparecerá abaixo.
    *   Por padrão, todos vêm com presença ("P" - verde).
    *   Clique no botão "P" para alternar para Falta ("F" - vermelho) nos alunos ausentes.
4.  Clique no botão **"ENVIAR CHAMADA"** ao final da lista para salvar.
    *   *Dica: Você receberá uma confirmação de sucesso.*

#### B. Como lançar Notas
1.  No cartão "Notas", selecione a **Disciplina**.
2.  A lista de alunos carregará com os campos de nota (Nota 1, Nota 2, etc, conforme configurado).
3.  Digite as notas (use ponto ou vírgula para decimais).
4.  Clique em **"SALVAR NOTAS"** para gravar.

## 3. Perfil: COORDENADOR

### Dashboard de Gestão
O coordenador tem uma visão completa da unidade/curso.

#### Ações Rápidas (Botões Coloridos Esquerda)
*   **+ Novo Aluno:** Abre formulário para cadastrar novo aluno (Nome, CPF, E-mail, Matrícula).
*   **+ Novo Professor:** Cadastro de novos docentes.
*   **+ Nova Disciplina:** Criação de matérias para a grade curricular.

#### Gestão de Alunos (Centro-Topo)
*   **Seleção:** Busque um aluno pelo nome na caixa de seleção.
*   **Botão PDF:** Gera a Ficha Cadastral do aluno selecionado.
*   **Botão Editar:** Abre janela para alterar dados do aluno (corrigir nome, senha, etc).

#### Desempenho por Turma
*   Lista todas as disciplinas do curso.
*   Mostra média da turma e contagem de aprovados/reprovados.
*   **Botão "Ver Notas":** Abre modal para editar notas de qualquer aluno daquela turma.
*   **Botão "Gerar Relatório PDF":** Baixa ata de resultados da disciplina.

#### Relatórios e Frequência (Botões Especiais)
Localizados na coluna de ferramentas:
1.  **ABRIR DIÁRIO DE CLASSE:**
    *   Permite ao coordenador lançar presença em nome de um professor.
    *   Funciona igual ao painel do professor.
2.  **RELATÓRIOS DE FREQUÊNCIA:**
    *   Abre um painel geral com todas as disciplinas.
    *   Mostra: Total de Aulas Dadas, % de Presença Média.
    *   **Botão "Alunos" (Novo):** Clique para ver a lista de alunos daquela disciplina específica, com a frequência individual de cada um (Ideal para conferência final).

#### Comunicação (Direita)
*   **Requerimentos Inbox:** Lista solicitações dos alunos. Clique para marcar como "Resolvido".
*   **Novo Comunicado:** Envie mensagens que aparecerão no mural de todos os alunos.
